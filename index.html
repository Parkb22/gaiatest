This is a simple 1 page light-weight >300 line tool meant to automate API calls, perfect for testing domain throughput across different domains on Gaianet. Everything is done in your browser, and you can verify the code using the github link below

Once you have started the script: Visit Gaia Credits on the Gaianet Dashboard and ensure that your credits are being consumed. If your credits aren't being consumed, you are NOT earning points.
[?] Use format subdomain.gaia.domains; do not include http:// or https:// or anything else
The default domain is configured for high API calls whereas many custom domains are not. Leaving this blank may result in higher throughput. Entering a custom domain will alternate API calls with a high-throughput server to ensure stable token usage.
You can generate a 1 time API key then delete it when you are done using the tool.
To generate an API key, visit: https://www.gaianet.ai/setting/gaia-api-keys
Be sure to top up your Gaia Credits. You can redeem points for credits here: https://www.gaianet.ai/reward-summary
You can redeem 500 Credits every 24 hours.
This script takes approximately 2.5 hours to fully use up 500 credits.
It is recommended to run this tool once every day to ensure maximum points
